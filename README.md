# CarryOn

개인 태스크 관리 웹앱. 칸반 보드로 태스크를 관리하고 캘린더로 날짜별 흐름을 확인합니다.

## 기능

### 칸반 보드
- 컬럼 추가/삭제/이름 변경, 카드 드래그로 순서·컬럼 이동
- **필터 컬럼** — 마감일·생성일·완료일 기준으로 N일 범위 태스크를 자동으로 표시 (드롭 불가)
- 태스크 추가 시 컬럼 최상단에 삽입
- 완료 체크버튼으로 완료 처리 (낙관적 UI)
- 다음 컬럼으로 이동 버튼 (모바일)

### 태스크 관리
- 제목, 메모, 마감일, 컬럼 편집
- 마감일 초과 시 빨간색, D-Day 표시
- 완료 시 날짜 기록, 보드에서 숨김 / 검색 시 표시

### 캘린더
- 월간 그리드, 이전/다음 달 미리보기 (클릭 시 해당 달 이동)
- 날짜 클릭 시 생성·완료·경유 태스크 확인
- 공휴일·일정·휴가 등록 및 삭제
- 날짜에서 바로 태스크 추가

### 계정
- Supabase 이메일 인증
- 최근 로그인 계정 5개 기억 (클릭 시 자동 로그인)
- 설정 모달: 로그아웃, 전체 초기화 (이메일 확인 후)
- PWA 설치 지원

## 기술 스택

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- @dnd-kit/core, @dnd-kit/sortable
- Supabase (Auth + PostgreSQL + RLS)
